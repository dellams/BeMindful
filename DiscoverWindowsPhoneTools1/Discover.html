Windows Phone SDK 8.0

Get started building apps for Windows Phone 8 or Windows Phone 7.5 with Windows Phone SDK 8.0.

Windows Phone SDK 8.0 extends Visual Studio 2012 to give you the tools you need to help you develop apps for Windows Phone, including Windows Phone app and library templates, Windows Phone Emulator, and performance analysis tools. You can use this Visual Studio extension to compile and debug your Windows Phone apps, and then deploy them to a Windows Phone or Windows Phone Emulator.
